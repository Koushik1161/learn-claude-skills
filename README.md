# Learn Claude Skills

A fun, interactive classroom game to learn **Claude Agent Skills** - the SKILL.md system that lets you create reusable instruction packages for Claude.

## Play Now

Just open `index.html` in your browser - no installation needed!

**Controls:**
- **Arrow Keys / WASD** - Move around
- **Space** - Interact / Continue
- **ESC** - Close dialogs

## How to Play

1. Walk around the classroom
2. Visit each station to **learn** about Skills
3. Take a **quiz** after each lesson
4. Complete all 4 stations to graduate!

### Stations

| Station | Topic |
|---------|-------|
| Whiteboard | Skill basics |
| Bookshelf | SKILL.md structure |
| Computer | Tools & integration |
| Study Desk | Advanced tips |

## What You'll Learn

- What Claude Skills are and why they matter
- How to write a SKILL.md file
- YAML frontmatter structure
- How Skills work with Claude's tools
- Best practices and tips
